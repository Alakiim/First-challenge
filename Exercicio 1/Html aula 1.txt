Funções:

class - classificação usada pra outra parte no javascrip
contenteditable - permite modificiar o testo no preview mas quando salva perde a modificação
data-(+qualquer nome separado por -) - usado no javascrip
hidden - esconde a tag
id - um por pagina que usa no javascrip ou css
style -> vc personaliza como troca de cores
Tabindex="" -> é onde e a ordem q a tecla tab pula

<p></p> -> é para escrever parágrafos
<em></em>  -> é para destacar uma escrita (italico)
<strong></strong> ou <b></b>  -> é para colocar em negrito
<div>  -> agrupa elementos como parágrafos e etc
<br> -> (pode ser </br> ou <br/> e fecha automatico) é para quebrar a linha
<h1></h1>  -> Cria um titulo na pagina
<h2></h2>  -> Cria um subtitulo  h3 h4 h5 h6 tbm quanto maior numero menor o titulo
<title></title> -> Cria um titulo na aba do navegador
<li></li>  -> Cria uma lista com uma ° no começo (se digitar li*um numero coloca a quantidade de itens de uma vez)
<ol></ol>  -> Cria uma lista ordenada com numeração no começo
<ul></ul>  -> Cria uma lista não ordenada por numero e sim com a ° 
<abbr tittle="texto">palavra</abbr> -> cria uma anotação visivel ao descansar o mouse sobre a palavra
<address>   -> tag pra marcar contato
<button> -> cria botoes

Caracteres reservados: 
&nbsp;  -> usa para dar espaço sem ser ignorado pelo html
&lt; = lower than = menor que
&gt; = greater than = maior que
&quot;  = aspas " 

<!DOCTYPE html>    -> usado no inicio de todo html
<html>
	<head>

	</head>

	<body>

	</body>
</html>
